Chapter A4
Reporting bugs

For tracking problem reports and bugs, we use GNATS, which is a bug tracking system published under the GNU license. We have set up a web interface at http://www.cactuscode.org/BugReporting, which allows easy submission and browsing of problem reports.

A description of the GNATS categories we use is provided in the Appendix D6.
